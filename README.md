# Временное название: RoleMaster

### Описание:
RoleMaster - это браузерная игра в жанре визуальной новеллы. Главный герой проходя различные главы сюжета встречается с гидами, которые помогают ему в прохождении каждой главы. <br>
Для Администраторов открываются возможности внесения изменений в зарегистрированные аккаунты, а также увидеть все существующие товары, обращения, блюда и NPC

### Команда разработчиков:
- alowe17 - backend разработчик, отображение данных на сайте, работа с БД
- нейросети - дизайн, анимации на сайте

### Инструкция запуска:
1. Копируем весь проект.
2. Создаем базу данных mysql с названием game_db_1 или можете использовать свою, но в таком случае поменяйте название в файле `application.properties`.
3. Запускать необходимо на версии Java 21.
4. Запускаем WebServiceApplication.
5. Переходим по ссылке: http://localhost:8080/login.
6. Если необходимо, то регистрируемся и уже после входим в аккаунт.

### 1.1.0 Обновление
- Добавлена страница поддержка
- Все обращения отправленные в поддержку отображаются в админ-панели самой последней таблицей

- Добавлена папка `admin` в `templates`, в которой хранятся все страницы, что относятся к админ-панели: `admin`, `changePassword`, `updateUser`
- Немного отредактирован текст на сайте
- Добавлена страница сюжет с главами, которым игрокам предстоит пройти
- Добавлена сущность `MessageSupport`
- Добавлен Enum status для обращений.
  - NEW - новое обращение.
  - REJECTED - отмененное обращение.
  - IN_PROGRESS - обращение находится на рассмотрении.
  - CLOSED - обращение закрыто с ответом.
